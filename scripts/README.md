# `scripts`

This directory is for one-off helper scripts and patch utilities that are useful during rapid prototyping, migration experiments, or content passes.

## Current scripts

- `patch-voices.py` — historical patch helper for voice-related changes in `packages/backend/convex/game.ts`
- `patch-unchosen.py` — historical patch helper for unchosen-self related edits in `packages/backend/convex/game.ts`

These are not part of the normal app runtime or setup flow. They live here so the repo root stays focused on the actual product structure.
